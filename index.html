What is HTML?

HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript, a programming language.

History

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. Another version, XHTML, was a rewrite of HTML as an XML language.

HTML Versions Timeline

HTML 2

HTML 2.0, published as RFC 1866 in November 24, 1995, defined the capabilities of HTML in common use prior to June 1994, creating a specification for interoperability among HTML implementations.

HTML 3

In January 14, 1997, an HTML 3.0 specification was released. Based upon earlier work on HTML+, extends HTML 2.0 to support tables, text flow around figures and math. This document has expired. It is no longer being maintained.

HTML 4

In December 18, 1997 HTML 4.0 was published as a W3C Recommendation. It offers three variations:

  • Strict, in which deprecated elements are forbidden
  • Transitional, in which deprecated elements are allowed
  • Frameset, in which mostly only frame related elements are allowed

HTML 5

HTML5 was first released in a public-facing form on 22 January 2008, with a major update and "W3C Recommendation" status in October 2014. Its goals were to improve the language with support for the latest multimedia and other new features; to keep the language both easily readable by humans and consistently understood by computers and devices such as web browsers, parsers, etc., without XHTML's rigidity; and to remain backward-compatible with older software. HTML5 is intended to subsume not only HTML 4 but also XHTML1 and even the DOM Level 2 HTML itself.

Elements

In HTML, an element is the fundamental building block of a webpage, consisting of a start tag, optional content, and an end tag, which together define a specific part of the document's structure and content.

Semantic HTML

Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the information in web pages and web applications rather than merely to define its presentation or look. Semantic HTML is processed by traditional web browsers as well as by many other user agents.
